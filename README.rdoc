= Welcome to Mconf-web
http://travis-ci.org/mconf/mconf-web.png {(more...)}[http://travis-ci.org/#!/mconf/mconf-web]

Mconf-web is a web application that controls the access to the Mconf webconferencing system.

It's developed in Ruby on Rails as a fork of {VCC}[https://github.com/ging/vcc].
Mconf-web integrates {BigBlueButton}[http://www.bigbluebutton.org/] and {Global Plaza}[http://globalplaza.org/] (VCC).

<b>For more details {read our wiki}[http://code.google.com/p/mconf/wiki/MconfWeb]</b>.
